# Twist - منصة الحلويات والمعجنات الأصيلة

Twist هو تطبيق ويب متكامل وشامل لبيع الحلويات والمعجنات بأسلوب عصري وأنيق (Artisanal). يركز التطبيق على تجربة المستخدم السلسة، والجمالية البصرية الراقية، وسهولة الإدارة لصاحب العمل.

## 🚀 المميزات الرئيسية

### لعملاء المتجر:
*   **واجهة مستخدم عصرية**: تصميم أنيق متجاوب مع كافة الشاشات (Mobile-first).
*   **تصفح ذكي**: إمكانية تصفح المنتجات حسب الفئات (حلويات، معجنات، مشروبات، إلخ).
*   **سلة تسوق تفاعلية**: إضافة وتعديل الكميات بلمسة واحدة مع حساب تلقائي للمجموع.
*   **تواصل مباشر**: زر واتساب عائم للتواصل الفوري مع المتجر.
    *   **نظام طلبات الواتساب**: عند إتمام الطلب، يتم إنشاء رسالة منسقة تلقائياً تحتوي على تفاصيل المنتجات والمجموع وعنوان العميل وإرسالها مباشرة لرقم صاحب المتجر.
*   **مشاركة الآراء**: نظام داخلي لإضافة التعليقات والمقترحات (Feedback).

### لنظام الإدارة (Dashboard):
*   **حماية بكلمة مرور**: لوحة تحكم مؤمنة للوصول للإعدادات (كلمة المرور الافتراضية: `1234`).
*   **إدارة المنتجات**: إضافة، تعديل، وحذف المنتجات بسهولة.
*   **إدارة الهوية البصرية**:
    *   تغيير شعار المتجر (Logo) عبر الرابط أو الرفع المباشر.
    *   تغيير صورة الهيرو (Hero Image) الرئيسية للمتجر.
*   **إدارة المخزون والبيانات**: إمكانية تصفير كافة البيانات والعودة للوضع الافتراضي.

## 🎨 الهوية البصرية (Visual Identity)

تم تصميم المتجر باتباع نمط **"Artisanal & Fine"**:
*   **الألوان**: باليت دافئ يعتمد على درجات البني الفاتح (Cream) والبرتقالي الهادئ (Accent).
*   **الخطوط**: مزيج بين خطوط Serif الفاخرة للعناوين وخطوط Sans-serif الواضحة للنصوص.
*   **التحريكات**: استخدام `framer-motion` و Tailwind animations لإضافة حيوية عند التمرير وتصفح المنتجات.

## 💻 التقنيات المستخدمة

*   **React 19**: لإدارة واجهة المستخدم بكفاءة عالية.
*   **TypeScript**: لضمان استقرار الكود وتقليل الأخطاء البرمجية.
*   **Tailwind CSS**: لتصميم سريع ومتسق مع دعم كامل للتجاوب.
*   **Lucide React**: لمكتبة الأيقونات العصرية.
*   **LocalStorage API**: لحفظ المنتجات، سلة التسوق، وإعدادات الهوية بشكل دائم في متصفح العميل.
*   **jspdf & html2canvas**: (اختياري) لدعم عمليات التصدير والطباعة.

## 📂 هيكلة الملفات

*   `/src/App.tsx`: المكون الرئيسي الذي يدير حالة التطبيق العامة.
*   `/src/components/`: يحتوي على المكونات الفرعية (Header, ProductCard, AdminDashboard, etc).
*   `/src/hooks/`: الخطافات المخصصة لإدارة المنتجات (`useProducts`) والسلة (`useCart`).
*   `/src/types.ts`: تعريفات الأنواع (Interfaces) والثوابت العامة.
*   `/src/constants.ts`: الثوابت الأساسية.

## 🛠️ كيف تبدأ؟

1.  **تثبيت الاعتماديات**:
    ```bash
    npm install
    ```
2.  **تشغيل المتجر في وضع التطوير**:
    ```bash
    npm run dev
    ```
3.  **البناء للإنتاج**:
    ```bash
    npm run build
    ```

## 📝 ملاحظات للمطورين

*   تم ضبط رقم الواتساب الافتراضي في ملف `types.ts`.
*   يمكن تغيير صورة الهيرو الافتراضية والشعار من خلال لوحة التحكم أو عبر تعديل ملف الثوابت.
*   التطبيق لا يحتاج إلى قاعدة بيانات خارجية (Backend) في نسخته الحالية، حيث يعتمد كلياً على التخزين المحلي، مما يجعله فائق السرعة وسهل النشر.
